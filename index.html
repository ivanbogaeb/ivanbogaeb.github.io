What is Steroid?
Steroid is a Wallpaper Engine web extension module that allows users to create their own customizable widgets, access news articles, connect to Spotify and AcuuWeather, recieve fully detailed computer metrics, create visualizers and execute programs within their wallpapers or desktop application powered by JavaScript, HTML and CSS.


Why?
This project has been heavily inspired by Rainmeter and the need to create a friendly JavaScript version that has all the latest web features, including cloud synchronization and API communication.


Made with:
HTML, CSS, Javascript, C#, .NET, Node.JS, Certbot, MongoDB, Redis, Electron, Express, Powershell and OpenHardwareLib.

What is Recharged?
Supercharged screen retransmition, fully customizable and flexible. Play games or connect to your PC from any device that supports a joystick USB/Bluetooth connection and HTML5.


Why?
One day, I was bored. I wanted to play some of my games on the TV, but wasn't able to. Nvidia GeForce Experience was a complete mess and having an AMD GPU forced me to create my own streaming application.


Made with:
Javascript, CSS, HTML, Python, Node.JS, Electron and Express.

What is Orthotics Schweiz?
Orthotics Schweiz is an orthotics and prosthetics factory/reseller based in Argentina, with more than 12 years of experience on the market and four stores open.


What did they need?
A website and stock management application was required. Cloud backup across their branch offices and data recovery mode in case of power shortages or unespected events. Also, it was imperative to manage all their products with a simple, modern application. Synchronizing all the data across their stores, keeping lifecycle and metrics for further market studies.


Made with:
HTML, CSS, JavaScript, PHP, MySQL, Electron.
